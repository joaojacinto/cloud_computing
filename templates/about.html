Sobre / Ajuda

Cloud Images App é uma aplicação desenvolvida para a disciplina de Computação na Nuvem. Permite aos utilizadores fazer upload, analisar e organizar imagens usando serviços Cloud da Google.

Serviços Cloud Utilizados:

  • Google Cloud Run: Deploy da aplicação Flask containerizada, escalável, sem necessidade de gestão de servidores.
  • Google Cloud Storage: Armazenamento das imagens enviadas, com links públicos diretos para download.
  • Google Cloud Firestore: Base de dados NoSQL para guardar os metadados das imagens, etiquetas e informações extra.
  • Google Cloud Vision API: Análise automática das imagens, detetando objetos e gerando etiquetas/labels.

Como Funciona:

  1. O utilizador faz upload de uma imagem pela interface web.
  2. A imagem é guardada no Cloud Storage.
  3. É feita a análise automática com a Cloud Vision API (labels + confiança).
  4. Os resultados e metadados (nome, data, tamanho, labels, etc) são guardados no Firestore.
  5. No dashboard podes ver todas as imagens, fazer download, consultar detalhes, e apagar registos.

Funcionalidades:

  • Upload de imagens e análise automática
  • Dashboard com detalhes (labels, confiança, data, tamanho, etc.)
  • Download direto das imagens
  • Paginação (para grandes volumes de imagens)
  • Apagar imagens do Storage e do Firestore
  • Confirmação visual de sucesso/erro (SweetAlert2)

Arquitetura (resumo):

  • Frontend: Flask + HTML/CSS/Bootstrap + SweetAlert2
  • Backend: Python Flask, Google Cloud SDKs
  • Infraestrutura: Container Docker na Google Cloud Run

Ajuda / Suporte:

  • Se encontrares erros, garante que tens ligação à internet.
  • Certifica-te que as permissões da conta de serviço GCP estão corretas.
  • Para dúvidas técnicas, consulta a documentação dos serviços Google Cloud ou contacta o responsável do projeto.
